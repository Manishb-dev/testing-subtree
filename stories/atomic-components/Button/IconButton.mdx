# IconButton

The `IconButton` component is a thin wrapper around [MUI IconButton](https://mui.com/material-ui/react-button/#icon-button), inheriting all its props and behaviour while extending it with a `variant` prop and custom `extraLarge` size - themed via the Tonic design system.

---

## Usage

```tsx
import { IconButton } from '@bmi/mui-tonic-components';

<IconButton color="primary" size="medium">
  <MyIcon />
</IconButton>
```

---

## Variants

Tonic IconButton supports two visual variants:

### Default
Transparent background with a colored icon. Use for low-emphasis icon actions.

### Contained
Filled circular background using `color.main`, white icon. Use for high-emphasis icon actions.

## Sizes

Tonic adds an `extraLarge` size on top of MUI's defaults. All four sizes use a 24 px icon; only the padding differs.

| Size | Total size | Padding |
|------|-----------|----------|
| `small` | 32 px | 4 px |
| `medium` | 36 px | 6 px |
| `large` | 40 px | 8 px |
| `extraLarge` | 48 px | 12 px |
