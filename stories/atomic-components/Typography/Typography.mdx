# Typography

Renders text using the Tonic theme's type scale. Includes all standard MUI variants plus three Tonic extensions: `display`, `body3`, and `body4`. All font families, sizes, weights, and line-heights are defined in `@bmi/mui-tonic-theme`.

---

## Usage

```tsx
import { Typography } from '@bmi/mui-tonic-components';

<Typography variant="h3" color="text.primary">
  Page heading
</Typography>
```

## Type Scale

All variants in the Tonic type scale. Tonic extends MUI's defaults with `display`, `body3`, and `body4`.

| Variant | Size | Weight | Line-height |
|---|---|---|---|
| `display` | 56px | 700 | 64px |
| `h1` | 40px | 700 | 48px |
| `h2` | 32px | 700 | 40px |
| `h3` | 24px | 700 | 32px |
| `h4` | 20px | 700 | 28px |
| `h5` | 16px | 700 | 20px |
| `h6` | 14px | 700 | 20px |
| `body1` | 20px | 400 | 32px |
| `body2` | 16px | 600 | 24px |
| `body3` ✦ | 14px | 400 | 24px |
| `body4` ✦ | 12px | 400 | 20px |

✦ Tonic extension - not in MUI's default type scale.

## Under the Hood

This component wraps [`MuiTypography`](https://mui.com/material-ui/api/typography/).

**MUI base:** `@mui/material/Typography`  
**Theme override:** `packages/theme/src/foundations/typography.ts`  
**Type augmentation:** `display`, `body3`, `body4` declared in `packages/theme/src/augmentation.ts`
