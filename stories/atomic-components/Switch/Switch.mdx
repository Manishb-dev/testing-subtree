# Switch

A Tonic wrapper around [MUI Switch](https://mui.com/material-ui/react-switch/) that enforces the Tonic size vocabulary and provides integrated label support via `FormControlLabel`.

---

## Usage

```tsx
import { Switch } from '@bmi/mui-tonic-components';

<Switch color="primary" size="medium" />
```

## Under the Hood

This component is a thin wrapper around [`MuiSwitch`](https://mui.com/material-ui/api/switch/).

**MUI base:** `@mui/material/Switch`  
**Tonic override file:** `packages/theme/src/overrides/switch.ts`

All visual styling - track shape, thumb size, colors, padding - lives in the theme override, not in the component. The component's only job is to enforce the Tonic prop vocabulary and forward everything else to MUI.

## Accessibility

- Always pair a switch with a visible label - use the `label` prop or a sibling `<label>` element so the purpose is clear.
- When used without a label (e.g. in a settings table row), provide `inputProps={{ 'aria-label': '…' }}` to describe what the switch controls.
- The switch communicates its on/off state via `aria-checked` automatically - do not duplicate this in visible text.
- Disabled switches retain their role in the accessibility tree but are non-interactive; ensure surrounding context still explains the feature.
- All Tonic theme colors meet WCAG AA contrast requirements for the track against a white background.

---

## Do / Don't

**Do**
- Use a switch for settings that take immediate effect without requiring a confirm action (e.g. "Enable dark mode").
- Use `color="error"` to indicate a toggle that activates a destructive or high-risk feature.
- Pair with a concise label that describes the effect in the on state (e.g. "Receive email notifications", not "Notifications").

**Don't**
- Don't use a switch for multi-step forms where the change needs explicit submission - use a Checkbox instead.
- Don't control `checked` without also wiring `onChange` - a controlled switch with no handler locks the UI.
- Don't use more than one `color` palette per group of related switches - keep a consistent color within a settings section.
