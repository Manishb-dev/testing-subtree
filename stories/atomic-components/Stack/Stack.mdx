# Stack

A one-dimensional layout component that arranges children in a row or column with uniform spacing. Spacing values map to the Tonic theme's 8px spacing unit (`spacing={1}` = 8px, `spacing={2}` = 16px, etc.).

---

## Usage

```tsx
import { Stack } from '@bmi/mui-tonic-components';

<Stack direction="row" spacing={2} alignItems="center">
  <Item />
  <Item />
  <Item />
</Stack>
```

## Directions

Stack supports four `direction` values. Use `row` for horizontal layouts and `column` for vertical ones. The `-reverse` variants flip the order without changing the DOM structure.

## Spacing Scale

Spacing maps to `theme.spacing(n)` - multiples of the base 8px unit. Use whole and half values from the scale for consistency.

## Under the Hood

This component wraps [`MuiStack`](https://mui.com/material-ui/react-stack/).

**MUI base:** `@mui/material/Stack`  
**Theme override:** none - all styling is done through props and `sx`.
