# FedExBanner

Promotional banner for the Licensing Portal that surfaces a FedEx shipping discount offer to BMI members. Renders a wide desktop layout with a tear-off promo code and automatically collapses to a stacked mobile layout when the container width drops below 640 px.

---

## Usage

```tsx
import { FedExBanner } from '@bmi/tonic-components-licensing';

<FedExBanner
  title="Save up to 80% on FedEx® Services"
  desc="New, bigger discounts designed for small business budgets exclusively for BMI members."
  code="FJHZNC"
  infoTooltip="Available to active BMI members only."
  onRedeem={() => handleRedeem()}
/>
```

## Accessibility

- The promo code copy button announces the copy action visually via an icon swap (`CopySimpleIcon` → `CheckIcon`) and resets after 2 seconds.
- The info icon wrapper has `tabIndex={0}` so keyboard users can focus and trigger the tooltip.
- The logo `<img>` carries `alt="logo"` — override the `logo` prop only with images that have meaningful alternative text in their context.

---

## Do / Don't

**Do**
- Let the banner manage its own layout — do not constrain it to a fixed pixel width that prevents the mobile breakpoint from triggering.
- Pass a translated `title` and `desc` when the consuming app supports multiple locales.

**Don't**
- Don't import `FedExBannerMobile` directly; it is an internal sub-component and its API may change.
- Don't use this component outside the Licensing Portal — it contains licensing-specific branding and is not part of the shared Tonic library.
