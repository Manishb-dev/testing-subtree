# How to use the packages

Step-by-step guides for consuming Tonic components in your React application.
Choose the framework that matches your project — **MUI** for standard web apps, **Tamagui** for cross-platform.

---

## MUI (Web)

### 1. Install dependencies

<Source language="bash" code={`npm install @bmi/tonic-components-mui @bmi/tonic-theme-mui @bmi/tonic-icons-web \\
  @mui/material @emotion/react @emotion/styled`} />

### 2. Apply the Tonic theme

Wrap your application root with MUI's `ThemeProvider` and pass in the Tonic theme.
This ensures every Tonic component picks up the correct colours, typography and spacing.

<Source language="tsx" code={`// src/App.tsx
import { ThemeProvider, CssBaseline } from '@mui/material';
import { lightTheme } from '@bmi/tonic-theme-mui';

export default function App() {
  return (
    <ThemeProvider theme={lightTheme}>
      <CssBaseline />
      {/* your application routes / pages */}
    </ThemeProvider>
  );
}`} />

> **Tip:** Tonic also exports a `darkTheme` if your app supports dark mode.

### 3. Use components

<Source language="tsx" code={`import { Button } from '@bmi/tonic-components-mui';

export function MyPage() {
  return (
    <Button variant="contained" onClick={() => console.log('clicked')}>
      Pay Now
    </Button>
  );
}`} />

### 4. Use icons

Icons live in their own package. Import them from `@bmi/tonic-icons-web`:

<Source language="tsx" code={`import { StarIcon } from '@bmi/tonic-icons-web';

export function MyPage() {
  return <StarIcon size="medium" />;
}`} />

## Tamagui (Cross-platform)

### 1. Install dependencies

<Source language="bash" code={`npm install @bmi/tonic-components-tamagui @bmi/tonic-theme-tamagui \\
  tamagui @tamagui/config react-native-web`} />

### 2. Apply the Tonic Tamagui theme

Wrap your application root with `TamaguiProvider` and pass in the Tonic Tamagui config:

<Source language="tsx" code={`// src/App.tsx
import { TamaguiProvider } from 'tamagui';
import { tamaguiConfig } from '@bmi/tonic-theme-tamagui';

export default function App() {
  return (
    <TamaguiProvider config={tamaguiConfig} defaultTheme="light">
      {/* your application routes / pages */}
    </TamaguiProvider>
  );
}`} />

### 3. Use components

<Source language="tsx" code={`import { Button } from '@bmi/tonic-components-tamagui';

export function MyPage() {
  return (
    <Button variant="contained" onPress={() => console.log('clicked')}>
      Pay Now
    </Button>
  );
}`} />

---

## Key Rules

1. **MUI consumers:** always import from `@bmi/tonic-components-mui` — never directly from `@mui/material`.
2. **Tamagui consumers:** always import from `@bmi/tonic-components-tamagui` — never directly from `tamagui`.
3. **Never add your own `ThemeProvider` inside a page or component** — the root provider is sufficient.
4. **Icons must be imported from `@bmi/tonic-icons-web`** — they are not re-exported from the component packages.
5. **All visual styling lives in the theme** — update `@bmi/tonic-theme-mui` or `@bmi/tonic-theme-tamagui`; do not add inline overrides.
