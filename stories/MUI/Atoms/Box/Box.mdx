# Box

A generic layout primitive that renders a `<div>` by default. It accepts all MUI `sx` system props, making it the foundation for one-off layout needs that do not warrant a dedicated component. All design tokens (spacing, color, border-radius, shadow) come from the Tonic theme.

---

## Usage

```tsx
import { Box } from '@bmi/tonic-components-mui';

<Box sx={{ p: 2, bgcolor: 'background.paper1', borderRadius: 1 }}>
  Content
</Box>
```

## Under the Hood

This component wraps [`MuiBox`](https://mui.com/material-ui/react-box/).

**MUI base:** `@mui/material/Box`  
**Theme override:** none - all styling is done by the consumer via `sx`.

The component's only job is to forward `sx` as an array (enabling merge-safe composition) and pass everything else through to MUI.
