# ListItem

Interactive list entry for navigation menus, selection lists, and option panels. Supports primary text, secondary text, avatars, end icons, and checkboxes through composable props.

---

## Usage

```tsx
import { ListItem } from '@bmi/tonic-components-mui';

<ListItem primaryText="Account Settings" onClick={() => navigate('/settings')} />
```

## Under the Hood

This component composes [`MuiListItemButton`](https://mui.com/material-ui/api/list-item-button/), [`MuiListItemText`](https://mui.com/material-ui/api/list-item-text/), [`MuiListItemAvatar`](https://mui.com/material-ui/api/list-item-avatar/), and [`MuiListItemIcon`](https://mui.com/material-ui/api/list-item-icon/).

**MUI base:** `@mui/material/ListItemButton` + layout primitives
**Theme override:** `packages/theme/src/overrides/list.ts`

All visual styling lives in the theme override. The component's job is to orchestrate the correct MUI sub-components based on the provided props.
