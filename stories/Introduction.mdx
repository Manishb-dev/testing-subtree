# Tonic - BMI Design System

**Version:** 0.1.0 | **Built on:** MUI v5 | **Status:** In Development

---

## What is Tonic?

Tonic is BMI's internal design system built on top of Material UI (MUI).
It provides a shared component library, design tokens, and documentation
used across the Licensing Portal and Affiliate Portal.

---

## Package Structure

| Package | Description |
|---|---|
| `@bmi/mui-tonic-theme` | MUI theme config, design tokens synced from Figma |
| `@bmi/mui-tonic-components` | Themed MUI component wrappers and shared patterns |
| `@bmi/mui-tonic-storybook` | This documentation site |

---

## How to use
```bash
npm install @bmi/mui-tonic-components
```
```tsx
import { Button } from '@bmi/mui-tonic-components';

<Button variant="contained">Pay Now</Button>
```

## Figma Sync

The theme is automatically synced from Figma variables via the `npm run theme:sync` command.
Any token changes in Figma are reflected in `@bmi/mui-tonic-theme` after the next sync.
