**Version:** 0.1.0 &nbsp;|&nbsp; **Built on:** MUI v7 &nbsp;|&nbsp; **Status:** In Development

---

## What is Tonic?

Tonic is BMI's internal design system.
It provides a shared component library, a Figma-synced theme, 45+ icons, and this
documentation site, all used across the **BMI**.

## Component Layers

Tonic organises components into three layers:

<table>
  <thead>
    <tr>
      <th>Layer</th>
      <th>Location</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Atomic</strong></td>
      <td><code>atoms/</code></td>
      <td>Typed wrappers around a single MUI component (Button, Select, Checkbox...)</td>
    </tr>
    <tr>
      <td><strong>Shared</strong></td>
      <td><code>shared/</code></td>
      <td>Composed from atomic components, reusable across portals (AmountEditField, AvatarLabel...)</td>
    </tr>
    <tr>
      <td><strong>Patterns</strong></td>
      <td><code>patterns/</code></td>
      <td>Full UI sections assembled from shared + atomic components</td>
    </tr>
  </tbody>
</table>
