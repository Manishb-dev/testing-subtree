# Tonic - BMI Design System

**Version:** 0.1.0 &nbsp;|&nbsp; **Status:** In Development

---

## What is Tonic?

Tonic is BMI's internal design system.
It provides a multi-framework component library, a Figma-synced theme, 45+ icons, and this
documentation site — all used across **BMI** products.

Tonic supports two UI frameworks side by side: **MUI** (web) and **Tamagui** (cross-platform).
Use the **Framework** dropdown in the toolbar to switch between them — only the relevant components
will appear in the sidebar.

## Component Layers

Tonic organises components into three layers — consistent across both MUI and Tamagui packages:

<table>
  <thead>
    <tr>
      <th>Layer</th>
      <th>Location</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>Atoms</strong></td>
      <td><code>atoms/</code></td>
      <td>Typed wrappers around a single framework component (Button, Select, Checkbox…)</td>
    </tr>
    <tr>
      <td><strong>Shared</strong></td>
      <td><code>shared/</code></td>
      <td>Composed from atoms, reusable across portals (AmountEditField, AvatarLabel, Stepper…)</td>
    </tr>
    <tr>
      <td><strong>Patterns</strong></td>
      <td><code>patterns/</code></td>
      <td>Full UI sections assembled from shared + atomic components</td>
    </tr>
  </tbody>
</table>
