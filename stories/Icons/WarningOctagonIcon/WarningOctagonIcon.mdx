# WarningOctagonIcon

Octagonal warning sign with an exclamation mark. Used for hard decline flows and critical error states.

---

## Usage

```tsx
import { WarningOctagonIcon } from '@bmi/tonic-icons-web';

<WarningOctagonIcon />
<WarningOctagonIcon size="large" />
<WarningOctagonIcon size="small" color="error" />
<WarningOctagonIcon variant="filled" />
<WarningOctagonIcon variant="contained" containedShape="circle" bgColor="error.main" />
```

## Variants

**Outlined** uses the stroke-weight paths (default). **Filled** uses the solid-weight paths for higher visual emphasis. **Contained** places the icon inside a shaped, colored background.
