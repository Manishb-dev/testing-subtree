# StarIcon

Used for ratings, favorites, and bookmarks. Accepts a `size` prop (px) and inherits `currentColor` from its parent.

---

## Usage

```tsx
import { StarIcon } from '@bmi/mui-tonic-components';

<StarIcon />
<StarIcon size={32} />
<StarIcon size={20} color="warning" />
```
