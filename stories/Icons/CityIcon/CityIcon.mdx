# CityIcon

CityIcon icon. Accepts `size`, `color`, `variant`, `containedShape`, and `bgColor` props.

---

## Usage

```tsx
import { CityIcon } from '@bmi/mui-tonic-icons';

<CityIcon />
<CityIcon size="large" />
<CityIcon size="small" color="primary" />
<CityIcon variant="filled" />
<CityIcon variant="contained" containedShape="circle" bgColor="primary.main" />
```

## Variants

**Outlined** uses the stroke-weight paths (default). **Filled** uses the solid-weight paths for higher visual emphasis. **Contained** places the icon inside a shaped, colored background.
