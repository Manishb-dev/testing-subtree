# CheckCircleIcon

Circle with a checkmark. Used for success and confirmation states. Accepts a `size` prop (px) and inherits `currentColor` from its parent.

---

## Usage

```tsx
import { CheckCircleIcon } from '@bmi/mui-tonic-components';

<CheckCircleIcon />
<CheckCircleIcon size={32} />
<CheckCircleIcon size={20} color="success" />
```
