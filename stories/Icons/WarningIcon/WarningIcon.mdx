# WarningIcon

Triangle with an exclamation mark. Used for caution and alert states. Accepts a `size` prop (px) and inherits `currentColor` from its parent.

---

## Usage

```tsx
import { WarningIcon } from '@bmi/mui-tonic-components';

<WarningIcon />
<WarningIcon size={32} />
<WarningIcon size={20} color="warning" />
```
