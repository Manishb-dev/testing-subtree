# PencilSimpleIcon

Pencil icon used for editing, writing, and signature contexts. Accepts `size`, `color`, `variant`, `containedShape`, and `bgColor` props.

---

## Usage

```tsx
import { PencilSimpleIcon } from '@bmi/mui-tonic-icons';

<PencilSimpleIcon />
<PencilSimpleIcon size="large" />
<PencilSimpleIcon size="small" color="primary" />
<PencilSimpleIcon variant="filled" />
<PencilSimpleIcon variant="contained" containedShape="circle" bgColor="primary.main" />
```

## Variants

**Outlined** uses the stroke-weight paths (default). **Filled** uses the solid-weight paths for higher visual emphasis. **Contained** places the icon inside a shaped, colored background.
