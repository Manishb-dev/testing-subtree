# CaretLineLeftIcon

Navigates to the first item or page in a sequence. Combines a chevron-left with a vertical bar. Use in pagination, sliders, and media controls to indicate a "jump to start" action.

---

## Usage

```tsx
import { CaretLineLeftIcon } from '@bmi/mui-tonic-icons';

<CaretLineLeftIcon />
<CaretLineLeftIcon size="large" />
<CaretLineLeftIcon size="small" color="primary" />
<CaretLineLeftIcon variant="filled" />
<CaretLineLeftIcon variant="contained" containedShape="circle" bgColor="primary.main" />
```

## Variants

**Outlined** uses stroke-weight paths (default). **Filled** uses solid paths for higher visual emphasis. **Contained** places the icon inside a shaped, colored background.
