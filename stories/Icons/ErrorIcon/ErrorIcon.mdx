# ErrorIcon

Circle with an exclamation mark. Used for error and danger states. Accepts a `size` prop (px) and inherits `currentColor` from its parent.

---

## Usage

```tsx
import { ErrorIcon } from '@bmi/mui-tonic-components';

<ErrorIcon />
<ErrorIcon size={32} />
<ErrorIcon size={20} color="error" />
```
